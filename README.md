# Homework Planner Website 

The Homework Planner App is a desktop tool designed to help students manage assignments, stay organized, and meet deadlines.
It features a GUI, Canvas API integration, a live clock, countdown timers, reminders, and full UI customization.

This project simulates what a real-world productivity application would do: organize tasks, sync with your LMS, and help you learn more effectively.
## Features
-Add, complete, and remove tasks
-View tasks in an organized list
-Automatic duplicate prevention
-SQLite-backed persistent storage
-Per-user task lists
-User registration & login
-Passwords hashed with PBKDF2
-Unique user accounts stored locally
