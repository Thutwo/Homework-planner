Homework Planner Website 

A simple and interactive Homework Planner Web App that helps students stay organized
by managing their assignments. 
Users can add, view, mark as completed, refresh, and remove homework tasks easily
keeping track of what's done and what's due. 

Features
- Add Assignments - Enter your homework tasks with ease.
- View Task - See all your current assignmetns at a glance.
- Refresh list - Instantly update or clear the displayed list.
- Mark as Done - Track completed homework by marking tasks as finished.
- Remove Tasks - Delete assignments once they're done or no longer needed
